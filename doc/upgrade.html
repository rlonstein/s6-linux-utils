How to upgrade s6-linux-utils

from 1.0.0 to 1.0.1

Bugfix release only. No change in dependencies.

from 0.12 to 1.0.0

Release 1.3.0 of skalibs is needed to build and/or run this version of s6-linux-utils.

FHS support has been added. See the end of this page for the rationale, and the general skarnet.org configuration page for how to proceed.

from 0.11 to 0.12

The build-time (and maybe run-time, if you are using shared libraries) dependencies have changed:

  • Version 1.2.4 of skalibs is required.

from 0.10 to 0.11

The build-time (and maybe run-time, if you are using shared libraries) dependencies have changed:

  • Version 1.2.3 of skalibs is required.
